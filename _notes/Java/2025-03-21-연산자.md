# 문제 1
```java
package operator.ex;  
  
public class OperationEx1 {  
    public static void main(String[] args) {  
        int num1 = 10;  
        int num2 = 20;  
        int num3 = 30;  
        int sum = num1 + num2 + num3;  
        int average = sum/3;  
        System.out.println(sum);  
        System.out.println(average);  
    }  
}
```
