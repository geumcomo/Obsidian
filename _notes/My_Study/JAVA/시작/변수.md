# 문제 1 
```java
package variable.ex;  
  
public class VarEx1Question {  
    public static void main(String[] args) {  
        int num1 = 4;  
        int num2 = 3;  
        System.out.println(num1 + num2 );  
        System.out.println(num1 - num2 );  
        System.out.println(num1 * num2 );  
    }  
}
```
